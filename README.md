# Enka.Network Kit for Swift

[Enka.Network](https://enka.network) API wrapper for Swift. Pretty much work in progress as of now.

### Supported Platforms

| Platform | Version |
| ------ | ------ |
| iOS | >= 15.0 |
| tvOS | >= 15.0 |
| watchOS | >= 8.0 |
| macOS (Catalyst) | >= 15.0 |
| macOS (Native) | >= 13.0 |
